# data/shared/

Общие ресурсы, используемые несколькими аккаунтами/моделями.

## Структура

```
shared/
├── source_videos/     # Исходные видео (движение тела) от оператора
│   └── <drive_id>.mp4
├── clothing/          # Одежда/образы для генерации в Nano Banana
│   └── <drive_id>.jpg
└── backgrounds/       # Фоны (если нужны)
    └── <name>.mp4
```

## Загрузка

Файлы загружаются автоматически из Google Drive при обработке задания (content_pipeline.py).
Кэшируются по Drive ID — повторная загрузка не нужна.

## Очистка

Старые файлы можно удалять вручную после публикации.
Размер: source_videos ~50MB/файл, clothing ~1MB/файл.
