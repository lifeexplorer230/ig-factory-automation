### Если ты ищешь...

**Общий план проекта**
```bash

# или
cat /root/.openclaw/workspace/shared/ig-roadmap.md | head -100
```

**Что делать на День 1**
```bash
grep -A 50 "День 1 (Пн)" /home/roma/ig-roadmap-checklist.md
```

**Команды для День 1**
```bash
grep -A 20 "День 1 (Пн)" /home/roma/ig-roadmap-quickstart.md
```

**Как работает Nano Banana API**
```bash
grep -A 30 "Nano Banana API" /home/roma/ig-roadmap-technical.md
```

```bash
ls -lh /home/roma/ig-roadmap* /home/roma/README.md /home/roma/START_HERE.md
```
